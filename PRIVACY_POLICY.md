# Privacy policy

## Core statement

This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it.

## What the app does

1. Streams microphone audio from the mobile app to the desktop app over local network (LAN/Wi-Fi) only, when the user starts streaming.
2. Performs local device discovery on the local network when the user taps/clicks discover.
3. Does not include built-in cloud telemetry, analytics, ad SDKs, or background upload of microphone data.

## External services

1. Release downloads are served by GitHub Releases when the user chooses to download installers/APKs.
2. Optional third-party components (for example VB-CABLE) are installed only if the user explicitly chooses to install them.

## Data control

1. Users can stop streaming at any time from either app.
2. Users can uninstall both apps using platform uninstall flows.
